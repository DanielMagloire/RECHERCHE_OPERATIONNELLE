L'éxécution du programme se fait par le fichier jar exécutable PRIMS.jar à partir du terminal. Ce dernier prend en arguments un fichier txt
Procédure d'exécution dans un terminal:
	1)se placer dans le repertoire contenant PRIMS.jar
	2) taper les commandes suivantes :

		$ cd Documents/RO/ (ici c'est pour se positionner dans le repertoire où se trouve le .jar à ex~cuter)
		$ java -jar PRIMS.jar test
		$ cat test (Pour afficher le contenu du fichier test)

		$ java -jar PRIMS.java nom_fichier_txt
